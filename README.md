# Pixora PHP Social App

Pixora is a PHP and MySql project 

Features:

- Login and registration
- Polished feed UI
- Image upload posts with captions
- Likes and comments
- Follow and unfollow users
- Profile page with stats and photo grid
- Account edit with avatar and bio
- Creator search

## Setup in XAMPP

1. Copy `pixora` into `C:\xampp\htdocs\`.
2. Start Apache and MySQL.
3. Open `http://localhost/phpmyadmin`.
4. Import `database.sql`.
5. Visit `http://localhost/pixora/register.php`.

Database config is in `config/db.php`.
